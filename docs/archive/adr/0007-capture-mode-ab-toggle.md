# MVP includes A/B capture modes: Standard (burst Keep-All) vs Max-Sensor (single shot)

Operators can toggle between Standard Capture Mode (~3 Keep-All shots at normal full quality) and Max-Sensor Capture Mode (one shot at maximum sensor resolution). Max-Sensor must not run a 3-shot Lean Burst — RAM and thermal cost make that combination unsafe. Capture Mode is an explicit selectable option; factory default is Standard.
