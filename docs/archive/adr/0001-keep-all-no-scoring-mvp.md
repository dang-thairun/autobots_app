# MVP delivers Keep-All Lean Bursts without on-device frame scoring

For each Passage we take a Lean Burst (~3 shots) and keep every Candidate Shot as a Kept Photo. Frame scoring (sharpness / smile / pose) is out of the default MVP path to minimize CPU, RAM, and complexity. Docs that describe “score then keep top 1” are superseded for MVP; ranking can return later as an optional mode.
